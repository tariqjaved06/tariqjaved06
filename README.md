### Electronics × Software

My BS Electronics & Computing background isn't a side note — it's the reason I'm heading toward AI/ML instead of staying purely web-focused. I work across:

- **Digital Logic & FPGA** — Verilog, Xilinx ISE, ModelSim
- **Signal Processing** — MATLAB
- **Circuit Design** — Multisim, Proteus, KiCad

Long-term, I'm interested in where these meet software: **Edge AI, AI-powered embedded systems, and intelligent hardware** — running models on the same kind of devices I already know how to design at the logic-gate level.
